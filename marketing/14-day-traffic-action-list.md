# Retro Vault Elite 14-Day Traffic Action List

This is the daily traffic plan. Do the actions in order. Keep it human, ask for feedback, and do not paste the same message everywhere.

Main site:
https://www.retrovaultelite.com/

Collector challenge:
https://www.retrovaultelite.com/collector-challenge.html

Promotion page:
https://www.retrovaultelite.com/promote.html

## Daily rules

- Post in no more than 2 communities per day.
- Message no more than 10 creators per day.
- Change the first sentence slightly if posting in more than one place.
- Reply to every real comment.
- If a moderator removes a post, do not repost it there.
- Ask for feedback first. Traffic comes second.

## Target list to build

Use these categories first:

- Small retro game collectors on YouTube with 1k to 50k subscribers.
- Retro pickup channels.
- Game room tour channels.
- eBay game hunting channels.
- CIB / sealed / graded game collectors.
- Local retro game shops.
- Retro gaming Discord servers.
- Facebook retro game collecting groups.
- Reddit communities that allow project or feedback posts.
- Product Hunt and Show HN for launch-style discovery.

Useful discovery searches:

- YouTube: `retro game collection tour`
- YouTube: `retro game pickups`
- YouTube: `SNES collection`
- YouTube: `game room tour retro games`
- YouTube: `CIB game collection`
- Google: `retro game collecting discord`
- Google: `retro gaming forum`
- Facebook: `retro game collecting`
